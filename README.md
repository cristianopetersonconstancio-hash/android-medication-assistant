# Android Medication Assistant

Projeto Android em desenvolvimento com foco em auxiliar pessoas idosas no controle de horários de medicamentos, reduzindo riscos de esquecimento ou ingestão incorreta.

## Objetivo
Fornecer alertas de medicação no horário correto, validar medicamentos por leitura de código de barras e disponibilizar informações essenciais para socorristas em situações de emergência, respeitando privacidade e segurança do usuário.

## Funcionalidades em desenvolvimento
- Alertas de horário para medicamentos
- Leitura de código de barras para validação da receita
- Prevenção de ingestão incorreta de medicamentos
- Tela de informações acessíveis para socorristas, mesmo com o dispositivo bloqueado

## Status
Projeto em desenvolvimento. Este repositório contém apenas partes do código e exemplos técnicos para fins de estudo e demonstração.
